Cartões de Memorização

Python

O que é o Python?

Python é uma linguagem de programação de alto nível, de propósito geral, interpretada e com uma sintaxe clara e legível. Ela foi projetada para ser fácil de aprender e usar, incentivando a programação orientada a objetos (OOP), além de permitir a programação imperativa e funcional. Python combina simplicidade com uma rica biblioteca padrão, tornando-o versátil para uma ampla gama de aplicações, desde scripts simples até sistemas complexos, como inteligência artificial e desenvolvimento web.

JavaScript

O que é o JavaScript?

JavaScript é uma linguagem de programação de alto nível e interpretada, amplamente usada para desenvolvimento web, tanto no lado do cliente quanto no lado do servidor (com Node.js). Ela permite adicionar interatividade e comportamentos dinâmicos a páginas web. JavaScript é conhecida por ser uma linguagem flexível, com tipagem dinâmica, e que pode ser usada em conjunto com HTML e CSS para criar experiências de usuário ricas.

Java

O que é o Java?

Java é uma linguagem de programação de propósito geral, orientada a objetos, e fortemente tipada. Ela foi projetada para ter o menor número possível de dependências de implementação, tornando-a altamente portável entre plataformas. Java é amplamente utilizada no desenvolvimento de aplicativos corporativos, dispositivos móveis (principalmente no desenvolvimento de Android), e sistemas distribuídos. Sua filosofia "escreva uma vez, rode em qualquer lugar" a tornou muito popular no desenvolvimento de software.

PHP

O que é o PHP?

PHP (Hypertext Preprocessor) é uma linguagem de programação amplamente usada para desenvolvimento web, especialmente para gerar conteúdo dinâmico em servidores. Ela é frequentemente integrada com bancos de dados, como MySQL, e usada para construir aplicações web robustas. Embora seja uma linguagem de script do lado do servidor, PHP pode ser misturado com HTML e JavaScript, permitindo o desenvolvimento rápido de páginas web dinâmicas e interativas.
